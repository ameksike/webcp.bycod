Elementos de configuración

            \$config['bycod']['view']['tpl']["html"] = "/src/client/html/";
            \$config['bycod']['view']['tpl']["php"]  = "/src/client/tpl/";

Las plantillas se definen por su extensión según el mecanismo que define la biblioteca Notary para su compilación. Por tanto se puede definir desde el archivo de configuración global la ruta para cada tipo de plantilla soportada por el motor de plantillas, en este caso serían HTML y PHP, tal y como se muestra en el ejemplo anterior.
